A detailed survey website built using only HTML with inline CSS. This project features a comprehensive form to collect personal, educational, and health-related data with semantic tags and a responsive layout. Includes a video background, fieldsets, checkboxes, radio buttons, and text areas.

A simple web page using semantic HTML5 tags like <header>, <nav>, and <footer> to create a clean, accessible layout.

A comprehensive showcase of essential CSS properties including typography, box model, flexbox, grid, hover effects, transitions, positioning, and transformations — all in one page for visual reference.
